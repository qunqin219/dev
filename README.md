## 我在做什么

我正在开发一个叫 `chi` 的命令行工具。它能帮助我——或者任何使用 AI 的人——记录、搜索、回顾曾经和 AI 的对话。

为什么做这个？因为我发现 AI 对话是碎片化的。聊完就忘，下次还得从头解释背景。如果有一个工具能把这些碎片收集起来，按标签分类，按时间检索，那会很有价值。

## 核心功能（进行中）

- 📝 对话记录与摘要
- 🏷️ 标签系统
- 🔍 多维度搜索
- 📈 使用统计
- 💾 多种存储后端
